API Title - MyFlix

A REST API for my web application that interacts with a database that stores data about different shows.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Returns a list of all movies /movies GET A JSON object that contains data about all movies
Returns data about a single movie by title /movies/:Title GET :title A JSON object containing data about the single movie requested
Returns a list of movies under the requested genre /movies/genre/:genreName GET :genreName A JSON object containing data about a specific genre of a movie
Returns data about a director by name /movies/directors/:directorName GET :directorName A JSON object containing data about a specific director
Allows new users to register /users POST A JSON object containing data to add, structured like: {"Username": "String","Password": "String", "Email": "String", "Birthday": "Date"} A JSON object that was added, structured like: {"Username": "String","Password": "String", "Email": "String", "Birthday": "Date"}
Allow users to update their information /users/:Username PUT :Username A JSON object containing the updated user information
Allow users to add movies to their list of favorites /users/:id/:favoriteMovies PUT :UserId A JSON object containing data about the user and what movie was added
Allow users to delete a movie(s) from their favorites list /users/:id/:favoriteMovies DELETE :UserId A JSON object containing data about the user and what movie was removed
Allow users to deregister /users/:Username DELETE :Username A JSON object containing data on the deleted user
